## Quickstart

```julia
using CryptoAPIs
using CryptoAPIs.Binance

Binance.Spot.ticker(;
    symbols = ["ADAUSDT", "BTCUSDT"]
)

Binance.Spot.candle(;
    symbol = "ADAUSDT",
    interval = Binance.Spot.Candle.M1
)

Binance.Spot.avg_price(;
    symbol = "ADAUSDT"
)

Binance.Spot.order_book(;
    symbol = "ADAUSDT"
)
```
